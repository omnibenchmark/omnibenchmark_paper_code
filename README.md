A clustering example for omnibenchmark

# How to run

1. Install omnibenchmark using [our tutorial](https://omnibenchmark.org/tutorial/) and activate the environment
2. `make` (Note that figures were built interactively, so this part was commented out of the `Makefile`)


## Aidan's porting of Open Problems

Likewise, here, figures were built interactively from the output text files, `aggregated_scores.csv` and `op_svg_results.json`. Code for the figures are still in `analyze_results_mark.Rmd`.
